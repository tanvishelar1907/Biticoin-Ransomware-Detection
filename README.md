# Bitcoin Ransomware Detection using Machine Learning

## Project Overview
This project uses machine learning techniques to detect ransomware-related Bitcoin transactions using the Bitcoin Heist dataset.

## Dataset Features
- year
- day
- length
- weight
- count
- looped
- neighbors
- income
- label

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Scaling
4. Model Training
5. Model Evaluation

## Results
The trained classification model identifies ransomware transaction patterns with good accuracy.
